# Invoice Management Application (MERN Stack)

This project is a **simple and professional Invoice Management Application** developed as part of a **MERN Stack interview assignment** for **DW Innovation Pvt. Ltd.**

The application allows users to securely log in, generate invoices, preview them in a GST-style format, and download invoice receipts.
